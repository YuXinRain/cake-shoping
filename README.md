## Synopsis

此專案為電商購物網站：<br>
使用者頁面提供了會員登入，會員註冊，購物車，結帳功能。<br>
管理者頁面提供了管理員登入，管理商品，管理訂單。<br>
專案架構的核心為React框架，使用了Redux來管理和更新應用程式狀態，使用了route來管理路由。
